## Installation

You can install the package via composer:

```bash
composer require javaabu-pvt-ltd/criminal-justice-sector-db-sdk
```

## Usage

```php
$skeleton = new Javaabu\CriminalJusticeSectorDB();
echo $skeleton->echoPhrase('Hello, Javaabu!');
```
